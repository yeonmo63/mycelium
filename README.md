# Mycelium (마이셀리움) - v1.0.0 🍄✨

**Mycelium**은 버섯 농장과 스마트 스토어의 데이터를 균사체처럼 촘촘하게 엮어 관리하는 지능형 통합 관리 시스템입니다. 
기존의 *CSI-MANAGER*에서 한 단계 진화하여, 모든 데이터를 **Product ID 중심의 이력 추적 체계**로 재편한 **Mycelium**의 정식 버전(v1.0.0)입니다.

---

### 🌟 Project Concept
본 프로젝트는 개발자의 직관과 AI의 정교한 분석이 결합된 **"바이브 코딩(Vibe Coding)"** 철학을 바탕으로 제작되었습니다. 단순히 기록하는 장부를 넘어, 데이터 간의 관계를 분석하고 비즈니스 통찰을 제공하는 것을 목표로 합니다.

---

## 🛠 Tech Stack
- **Core**: Rust (Reliable Backend)
- **Framework**: Tauri (Desktop App for Windows)
- **Frontend**: React (Modern & Dynamic UI)
- **Database**: PostgreSQL (SQLx with ID-based normalization)
- **AI Integration**: Gemini AI (Consultation & Data Analysis)

---

## 🚀 Key Features (v1.0.0 고도화)
- **ID-Based Product Lifecycle**: 상품의 생성, 이름 변경, 가격 인상, 병합 등 모든 생애 주기를 고유 ID로 추적합니다.
- **Smart Audit Logs**: 재고 수량의 단순 변화뿐만 아니라 상품 정보의 변화까지 타임라인으로 기록하는 지능형 로그 시스템을 제공합니다.
- **Advanced Stock Conversion**: 원자재와 완제품 간의 생산 전환 로직을 ID 기반으로 연동하여 재고 오차를 최소화했습니다.
- **Flexible Data Integrity**: 상품명 변경 시 과거 기록의 보존 여부를 사용자가 선택할 수 있는 유연한 데이터 정책을 지원합니다.

## 💻 How to Development
1. 의존성 설치: `npm install`
2. 개발 모드 실행: `npm run tauri dev`
3. 빌드: `npm run tauri build`
